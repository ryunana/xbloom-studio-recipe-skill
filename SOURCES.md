# Sources and Research Notes

This skill was prepared from public information and practical xBloom Studio brewing notes. Key source categories:

## Official xBloom / TBDx documentation

- xBloom Studio official product and landing pages
- xBloom Studio user manual / Help Centre articles
- xBloom Coffee App / recipe creation articles
- Three Creative Modes: Autopilot, Copilot, Freesolo
- Omni Dripper 2 guidance
- WAIT / overflow-protection troubleshooting
- xBloom Studio and Comandante C40 grind conversion table

## Public coffee community / review references

- Nucleus Coffee: xBloom Studio control variables
- Standout Coffee: xBloom Studio dial-in case study by Erik Persson
- Honest Coffee Guide: xBloom Studio grind settings
- Beean Coffee: xBloom Studio grind settings and micron reference
- BeanScene, Basic Barista, Barista Magazine, CoffeeGeek, and Whole Latte Love public xBloom Studio articles/reviews

## Notes

- This repository does not include xBloom proprietary assets.
- The C40 conversion values are summarized as public factual compatibility data.
- Recipes generated by the skill are best-practice starting points, not guaranteed final recipes; coffee age, roast style, water chemistry, filter paper, and grinder calibration can all change the result.
- This project is not affiliated with, endorsed by, or sponsored by xBloom.
