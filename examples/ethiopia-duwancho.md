# Example: Ethiopia Duwancho 74158 Natural

Input:

```text
ETHIOPIA DUWANCHO
埃塞俄比亚 朵望秋
74158 Natural
FLOWER BAYBERRY CRANBERRY ORANGE NECTARINE GREEN MANGO
花香 杨梅 蔓越莓 橙子 油桃 青芒
```

Recommended starting recipe:

- Dose: 15 g
- Water: 240 ml
- Ratio: 1:16
- Grind: 50, adjustable 48-52
- RPM: 110
- Pours: 45 / 65 / 65 / 65 ml
- Temperature: 93 / 93 / 92 / 92 °C
- Flow: 3.0 then 3.5 ml/s
- Pattern: Spiral / Spiral / Spiral / Circular
- Vibration: Bloom after ON only

Troubleshooting:

- Sour: grind 50 → 48
- Bitter/astringent: grind 50 → 52
- Thin: bloom pause 14s → 20s
